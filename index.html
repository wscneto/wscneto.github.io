Projects

Graph Algorithms

This is the implementation of 5 graph algorithms: Dijkstra, Bellman-Ford, Kosaraju, Kruskal and Prim. The project was crucial to consolidate my graph theory knowledge. It was written in C.

Bookaholic

Bookaholic is an online bookstore made with Vue.js, Python and MySQL. In this project, I put the Extreme Programming agile methodology into practice to satisfy customer's needs. I also implemented a microservices architecture, separating the program into 6 services: user, catalog, order, review, recommendation, and an API gateway. These were made using the Flask library.

Online Chess using TCP/IP

Chess is one of my favorite games ever. Why not practice network programming while having some fun? This is the idea behind my online chess game. It was made in Python, using the socket library. The program uses TCP protocol to connect client and server using their IP addresses.
